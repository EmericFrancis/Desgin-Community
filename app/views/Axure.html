Axure

To enable running of Axure RP Pro for wireframing and prototpes

Apply for licence

  • To request installation of the software you will need to raise a Request for Service via 'MySelfService'.

  • Go to the 'Start' menu and select 'All Programs'. 'MySelfService' is under #F6 MY REQUEST HUB'.

  • You will then see the main option screen of the mySelfService. Click on 'Standard IT request' and then type 'software' to find the 'Software install/ Renew/ Removal or Transfer Request' option.

  • Click on this option, fill out your details and click 'Raise Request'.

  • You will receive an email with the request number which you can use in the Marvel system to check progress of your request.

Download Axure

  • Go to www.axure.com and click on the center 'Download Free Trial' option.

  • This will download a 30 day free trial for you to install on the Macbook whilst you wait for the RFS to complete.

  • Once your RFS has provided you with a licence key and licensee name. Open Axure and open 'Help - Manage licence'.

  • Input the name and key and this will activate your version of Axure.

GDS Widget library

  • To install the Axure library containing GDS style widgets to use in your prototypes go to https://drive.google.com/drive/folders/0B_uf_jUFJvV_am1JdkRtalU5Z00

  • and download the GDS.rplib file. Then in Axure click the icon beside Libraries in the bottom left portion of the screen (shown below), click 'Load library' and navigate to the downloaded file. This will then add a 'GDS' option to the library dropdown options.

  • The GDS widget library is an active document so if you need a widget that is not present please create it and add it to the library for others to use.

  • Instructions for this are shown here - https://www.axure.com/support/reference/widget-libraries

New fonts

  • GDS and the Land registry both have their own fonts that you will need to install to use within Axure as well as in other applications. The GDS font is 'GDSTransportWebsite' and the Land registry font is 'GEO'.

  • To install the GDS font go to https://drive.google.com/drive/folders/0B1ZlxWkPbjnsMERzbUpjU2JwRTA

  • Download the GDSTransportwebsite-Bold.zip and the GDSTransportwebsite.zip files and unzip.

  • Then in each unzipped folder go into 'Fonts', double click on the .ttf file and click the 'install font' button, shown below.

  • To install the Land Registry font go to https://drive.google.com/drive/folders/0B1ZlxWkPbjnsNHBkTHBZaWY4b00

  • Download each .ttf file and install as above.

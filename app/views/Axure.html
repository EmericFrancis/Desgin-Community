Axure

To enable running of Axure RP Pro for wireframes and prototypes

Apply for licence

  • To request installation of the software you will need to raise a Request for Service via 'MySelfService'.
  • Go to the 'Start' menu and select 'All Programs'. 'MySelfService' is under #F6 MY REQUEST HUB'.
  • You'll then see the main option screen of the mySelfService. Click on 'Standard IT request' and then type 'software' to find the 'Software install/ Renew/ Removal or Transfer Request' option.
  • Click on this option, fill out your details and click 'Raise Request'.
  • You'll receive an email with the request number which you can use in the Marvel system to check progress of your request.

Download Axure

  • Go to www.axure.com and click on the center 'Download Free Trial' option.
  • This will download a 30 day free trial for you to install on the Macbook whilst you wait for the RFS to complete.
  • Once your RFS has provided you with a licence key and licensee name. Open Axure and open 'Help - Manage licence'.
  • Input the name and key and this will activate your version of Axure.

GDS Widget library

There is an Axure library containing GDS style widgets to use in your prototypes. This is an active document, so if you need a widget that is not present follow these instructions to create it so the rest of the community can use it as well.

  • Download the GDS.rplib file from the google drive.
  • In Axure, click the icon beside Libraries in the bottom left portion of the screen (shown below), click 'Load library' and navigate to the downloaded file. This will add a 'GDS' option to the library dropdown options.

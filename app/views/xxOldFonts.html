New fonts

  • GDS and HM Land registry both have their own fonts that you'll need to install to use within Axure as well as in other applications. The GDS font is 'GDSTransportWebsite' and the HM Land registry font is 'GEO'.

  • To install the GDS font go to https://drive.google.com/drive/folders/0B1ZlxWkPbjnsMERzbUpjU2JwRTA

  • Download the GDSTransportwebsite-Bold.zip and the GDSTransportwebsite.zip files and unzip.

  • Then in each unzipped folder go into 'Fonts', double click on the .ttf file and click the 'install font' button, shown below.

  • To install the HM Land Registry font go to https://drive.google.com/drive/folders/0B1ZlxWkPbjnsNHBkTHBZaWY4b00

  • Download each .ttf file and install as above.

Adobe creative cloud

To enable running of Adobe applications such as Illustrator and Photoshop

Apply for licence

  • To request installation of the software you'll need to raise a Request for Service via 'MySelfService'.

  • Go to the 'Start' menu and select 'All Programs'. 'MySelfService' is under #F6 MY REQUEST HUB'.

  • You'll then see the main option screen of the mySelfService. Click on 'Standard IT request' and then type 'software' to find the 'Software install/ Renew/ Removal or Transfer Request' option.

  • Click on this option, fill out your details and click 'Raise Request'.

  • You'll receive an email with the request number which you can use in the Marvel system to check progress of your request.

Download Applications

  • Once you have an account go to Adobe.com, create an account and sign in.

  • Then under 'Manage account' click 'Desktop Downloads'. From this screen download the 'Creative Cloud' desktop app.

  • This will then enable you to download and open any required applications, such as Illustrator and Photoshop more easily.

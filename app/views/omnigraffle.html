Omnigraffle

Get started with a digital illustration application.

Apply for licence

  • To request installation of the software you'll need to raise a 'Request for Service (RFS)' via 'mySelfService' on HMLR's intranet.
  • Go to the 'Start' menu and select 'All Programs'.'mySelfService' is under #F6 'MY REQUEST HUB'.
  • You'll then see the main option screen of the mySelfService. Click on 'Standard IT request' and then type 'software' to find the 'Software install/ Renew/ Removal or Transfer Request' option.
  • Click on this option, fill out your details and click 'Raise Request'.
  • You'll receive an email with the request number which you can use in the Marvel system to check progress of your request.

Download Omnigraffle

  • Download Omnigraffle
  • Once your RFS has completed you'll be provided with an exchange code that you will need to input.
  • You will then receive an email containing a personal licence key.
  • Open Omnigraffle and click 'Omnigraffle - Licences' to input this personal licence key, making your version of Omnigraffle official.

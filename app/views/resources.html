Fonts

When creating prototypes for externally facing services, the font GDS Transport Website should be used.

For all internal content Arial should be used.

Government Digital Services (GDS)

GDS have provided a Service toolkit to be used as a starting point when designing and building government services.

For us designers specifically, both the design system and prototyping kit should be referred to and used.

GDS also host a number of courses as well as cross-governmental events and training that may be useful.

Screenshoting

We all need to take screen shots from time to time, and there are some nifty tools out there to help.

  • Awesome screenshot is a google chrome extension for taking and annotating screenshots. There is also a screen video recorder, which can be useful.
  • Full page screen capture is a google chrome extension used to capture the entire page of your current browser window.
  • Nimbus is another full screen capture extension for chrome that can capture video's as well.
  • Andy held a design noodle session to tell us about WShot, a command line app to take screenshots of web page created by Joe Lanman.

Sketch

Sketch is mainly used to create service maps and journeys. It can be downloaded for free and a key requested through mySelfServe for its continued use.

It can also be used for creating wireframes and static prototypes. There is a sketch wireframing kit available to make this easier.

If you haven't used sketch before, it is worth checking out some tutorials.

Text editors

When prototyping in code, you are going to need an editor to work in. There are several available. Some commonly used at HM Land Registry include:

  • Atom is a text editor that's modern, approachable, yet hackable to the core - a tool you can customize to do anything but also use productively without ever touching a config file.
  • IntelliJ IDEA
  • Sublime text is a sophisticated text editor for code, markup and prose. You’ll love the slick user interface, extraordinary features and amazing performance.
  • TextWrangler's features include "grep" (regular-expression) pattern matching, search and replace across multiple files, function navigation and syntax coloring for numerous source code languages, code folding, FTP and SFTP open and save, AppleScript, Mac OS X Unix scripting support, and much more.

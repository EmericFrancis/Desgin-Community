Sketch

To enable running of vector graphics editor

Apply for licence

  • To request installation of the software you will need to raise a Request for Service via 'MySelfService'.
  • Go to the 'Start' menu and select 'All Programs'. 'MySelfService' is under #F6 MY REQUEST HUB'.
  • You'll then see the main option screen of the mySelfService. Click on 'Standard IT request' and then type 'software', in the search bar, to find the 'Software install/ Renew/ Removal or Transfer Request' option.
  • Click on this option, fill out your details and click 'Raise Request'.
  • You'll receive an email with the request number which you can use in the Marvel system to check the progress of your request.

Download Sketch

  • Download and install the free trial of sketch.
  • Once your RFS has provided you with a licence number, you can input this into Sketch by clicking on the 'Sketch' option in the top left of the screen and then the 'About and Registration' option to make your version of Sketch official.

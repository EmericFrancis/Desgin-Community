HMLR designer starter kit

Take some time to go through these steps. If you want any help - please ask any member of the community!

  1. 1 Apply for remote access

    Remote access will be required in order access to Office 365 from a non-diti computer.

  2. 2 Create accounts

    You will need to create accounts for Google, Trello, slack, github and heroku.

  3. 3 Request and install software

    The GOV.UK prototype kit is used to create prototypes. For medium fidelity prototypes, axure can be used.

    Other useful software include omnigraffle and sketch, either of which are used to create service maps or journey diagrams.

  4. 4 Training

    GDS host a number of courses as well as cross-governmental events and training that may be useful.

    The different applications that we use also have their own tutorials that may be useful.

  5. 5 Get included

    Join our community email distribution list to be kept in the loop. There are other communities across government that you might like to join as well.
